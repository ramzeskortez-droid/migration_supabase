# Version History

## 1.18.80 - Исправление CORS для AI
- **Backend (Edge Functions):**
  - Добавлена новая функция `process-email`. Она принимает текст заявки, отправляет его в Groq и возвращает JSON. Это решает проблему CORS.
- **Frontend:**
  - `AiAssistant` теперь вызывает `supabase.functions.invoke('process-email')` вместо `fetch('api.groq.com')`.
- **Security:**
  - `GROQ_API_KEY` полностью удален из клиентского кода. Теперь он должен храниться только в секретах Supabase.

## 1.18.79 - Валюта закупщика и защита роутов
- **Security:**
  - Внедрена защита маршрутов (/admin, /operator, /buyer) через `ProtectedRoute`. Прямой доступ по ссылке без токена теперь невозможен.
- **Logic:**
  - Исправлена валюта по умолчанию при создании оффера: теперь всегда **CNY** (ранее ошибочно ставилась RUB).
