# Version History

## 1.18.79 - Валюта закупщика и защита роутов
- **Security:**
  - Внедрена защита маршрутов (/admin, /operator, /buyer) через `ProtectedRoute`. Прямой доступ по ссылке без токена теперь невозможен.
- **Logic:**
  - Исправлена валюта по умолчанию при создании оффера: теперь всегда **CNY** (ранее ошибочно ставилась RUB).

## 1.18.78 - Интерактивный Калькулятор и Улучшенная Навигация
- **Manager Finance:**
  - Блок "Пример расчета" заменен на полноценный интерактивный **Калькулятор цены**.
  - Менеджер может ввести произвольную "Цену поставщика (¥)" и "Вес (кг)" для проверки итоговой стоимости по текущим курсам.
- **Manager Navigation:**
  - Реализован корректный переход к заказу из Глобального чата.
  - Исправлен цвет статуса "Ручная обработка" (фиолетовый).
- **Buyer Navigation:**
  - Исправлена проблема пустого поля поиска при переходе к заказу из чата (синхронизация `searchQuery`).
