# Version History

## 1.18.50 - Разделение данных: Source vs Draft
- **Buyer UI:** Реализовано четкое разделение данных в карточке товара.
  - `sourceItem`: Данные оператора (Read-only), приходят из БД (`details`).
  - `item`: Данные закупщика (Draft), приходят из стейта формы.
  - Это решает проблему отображения файлов оператора, даже если форма закупщика была очищена.
