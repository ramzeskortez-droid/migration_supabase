ВЕРСИЯ 1.18.31 - Мульти-загрузка файлов (PDF, Excel, Фото)

- **Database:** Добавлена колонка `order_files` (jsonb) в таблицу `orders` для хранения метаданных прикрепленных файлов.
- **Backend:** В `SupabaseService` реализована поддержка поля `order_files` при создании и получении заказов. Файлы загружаются в бакет `attachments` в папку `temp/`.
- **Operator UI:** 
  - Реализован новый блок "ФАЙЛЫ ПО ЗАЯВКЕ" с поддержкой Drag&Drop.
  - Успешно загруженные файлы отображаются в виде строки с гиперссылками через запятую прямо над областью загрузки.
- **Global UI:** В развернутом виде заказа (для Оператора, Менеджера и Закупщика) добавлена новая широкая строка в блоке основной информации, содержащая список прикрепленных файлов с гиперссылками.

БЫЛА ПРОБЛЕМА:
Оператор мог прикрепить только одно фото к каждой позиции, не было возможности прикрепить общие документы по заказу (инвойсы, спецификации в PDF/Excel).

КАК РЕШИЛИ:
Добавлена колонка order_files в таблицу orders. Создан компонент OrderFilesUpload с мгновенной загрузкой в Storage. Обновлены все компоненты детального просмотра заказа для отображения списка файлов в блоке основной информации.
