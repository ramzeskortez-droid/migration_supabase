ВЕРСИЯ 1.18.53 - Финализация UI/UX и исправление багов отображения
- **Buyer UI (Z-Index Fix):** Исправлено перекрытие выпадающего списка файлов элементами таблицы. Убран `overflow-hidden`, поднят `z-index` до 999.
- **Buyer UI (Sorting):** Сортировка списка по умолчанию изменена на "ID (по убыванию)", чтобы новые заказы всегда были сверху.
- **Buyer UI (Data):** Исправлен баг, при котором файлы оператора дублировались в форму "Ваши файлы" при создании нового оффера.
- **Admin UI:** Блок "Файлы от поставщиков" переименован в "Файлы от закупщиков".
- **Documentation:** В файл `bugs_log.md` добавлены подробные отчеты по инцидентам с файлами и UI.

БЫЛА ПРОБЛЕМА:
1. Выпадающее меню с файлами обрезалось границами карточки.
2. Новые заказы терялись в списке из-за сортировки по дедлайну.
3. Закупщик видел "чужие" файлы в своей форме отправки.
4. Терминология в админке ("поставщик" вместо "закупщик") путала менеджеров.

КАК РЕШИЛИ:
1. CSS фиксы (overflow/z-index).
2. Изменение дефолтного стейта `sortConfig`.
3. Разделение источников данных в `BuyerItemCard`.
4. Переименование в `AdminOrdersList`.
