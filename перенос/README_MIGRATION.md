# Инструкция по переносу рабочего окружения (AutoParts Marketplace Agent)

Этот архив содержит всё необходимое для развертывания агента и проекта на другом компьютере с сохранением всех настроек, прокси и MCP серверов.

## Состав архива
- `conductor/` — История задач и текущий план разработки.
- `.gemini/` — **КРИТИЧНО**: Все расширения (ComputerUse, Conductor), серверы MCP и глобальные настройки.
- `TG_BLOCK_V2.js` — Актуальный скрипт для Google Apps Script (Бэкенд).
- `start-gemini-api.bat` — Скрипт запуска с настройками прокси и путями.
- Конфигурационные файлы проекта (`package.json`, `vite.config.ts`).

## Пошаговая установка на новом ПК

### 1. Подготовка окружения
1. Установите **Node.js** (v18+).
2. Установите **Python** (v3.9+).
3. Установите **Google Chrome**.
4. Установите Gemini CLI: `npm install -g @google/gemini-cli`.

### 2. Перенос конфигурации (MCP)
1. Скопируйте папку `.gemini` из этого архива в корень вашего профиля пользователя:
   `C:\Users\<Ваше_Имя>\.gemini`
   *Если папка уже существует, объедините содержимое.*

### 3. Настройка проекта
1. Скопируйте папку проекта на диск.
2. В корне проекта запустите:
   ```cmd
   npm install
   ```
3. Установите зависимости для расширения ComputerUse:
   ```cmd
   pip install playwright pillow fastmcp
   playwright install chromium
   ```

### 4. Настройка запуска (start-gemini-api.bat)
Откройте `start-gemini-api.bat` в текстовом редакторе и исправьте пути под вашего пользователя:
1. Найдите путь к `python.exe` и замените `Dima` на ваше имя пользователя.
2. Найдите путь к `gemini.cmd` (обычно в `AppData\Roaming\npm`) и также исправьте имя пользователя.
3. Прокси уже настроен (161.115.231.113:9149), менять не нужно, если он активен.

### 5. Бэкенд (Google Sheets)
Если вы еще не обновили скрипт в таблице:
1. Откройте Google Таблицу > Расширения > Apps Script.
2. Замените код на содержимое `TG_BLOCK_V2.js`.
3. Перевыпустите деплой (New Deployment).

### 6. Запуск
Запускайте агента через `start-gemini-api.bat`. Он подхватит все расширения из перенесенной папки `.gemini`.
